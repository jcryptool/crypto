The Frequency Analysis

The Frequency Analysis basically counts the occurrence of all characters the text consists of. The text provided is visualized as a graph depicting the percental frequency of characters. Options to perform further analysis are given such as adding Vigenére parameters or a text overlay.

You can use this plug-in in two ways, depending on what you want to do with the frequency distribution:

1) The simple view (radio button "Simply give me a diagram") shows you a graph with the percental occurrences. You have the option to alter the text before analyzing it by clicking the button "Filter settings" (filter characters, transforming all characters to lower case, etc.).

The Properties section offers configuring the frequency analysis by choosing a monoalphabetic or polyalphabetic analysis. Monoalphabetic means that there is a single substitution over the whole text, while polyalphabetic means that there are multiple substitutions over the text. When choosing polyalphabetic, the option to configure Vigenére settings is given. The Key length can be configured which is the distance between repeated groups of cipher-letters, as multiple plaintext letters can correspond to a ciphertext letter in Vigenére ciphers. The offset allows to configure the offset of characters in the alphabet.

2) The extended view (radio button "Add more complex analysis abilities") offers more analysis abilities, which are located in the second tab in the properties area:

The Decipher and Overlay tab offers more analysis tools. The first option offered is to "overlay" a reference distribution. This will be semi-transparent, and can be useful in many ways. You can choose a reference text from which the distribution is calculated in the drop down menu below.

The next option is on which character set (alphabet) the reference distribution is displayed. If you only want to see what is the usual distribution of capital letters over a text as overlay, select the corresponding alphabet "Upper Latin".
Important: This option is adjusting itself correctly on ciphertexts normally.

The third tab labeled Loaded text shows information about the text: the name of the source, the number of characters, as well as a short excerpt.
