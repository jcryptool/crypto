The Chinese Remainder Theorem Algorithm

The Chinese Remainder Theorem Algorithm (CRT) is a solution method to solve simultaneous congruences. It will for example used to speed up the RSA algorithm, especially for the generation of digital signatures and decryption. Another application of the Chinese Remainder Theorem is to calculate the root of the Fiat-Shamir method.

How to use the according JCT plug-in is describes here.

We have linear equations (congruences) of the form

x ≡ a0 mod m1, x ≡ a2 mod m2, … x ≡ an mod mn

available with m1,…,mn natural numbers which are paired coprime and a1,…an integers. Now

m = ∏ mi, Mi = m∕mi, 0 ≤ i ≤ n

is calculated. There is a clear solution of the simultaneous congruences.It is the equation

gcd(mi,Mi) = 1, 0 ≤ i ≤ n

because of mi the values are paired coprime. We use extended Euclidean, to calculate the numbers yi ∈ ℤ,0 ≤ i ≤ n with

yiMi ≡ 1 mod mi, 0 ≤ i ≤ n.

Now we can use

x = (∑ aiyiMi) mod m 0 ≤ i ≤ n

and get a solution x of the simultaneous congruence. We assume the following theorem [1]

Theorem 1.1 Let m0,…,mn be paired coprime natural numbers and let a0,…an integer numbers. Then the simulantious congruences hat a solution x, which is clearly mod m = ∏ mi, 0 ≤ i ≤ n.

The proof of this can be seen in [1]. We want to show the algorithm with an example.

Example We want to solve the simultaneous congruence

x ≡ 2 mod 4, x ≡ 1 mod 3, x ≡ 4 mod 5, x ≡ 3 mod 7

So is m0 = 4, m1 = 3, m2 = 5, m3 = 7, a0 = 2, a1 = 1, a2 = 4, a3 = 3. Then is m = 420, M0 = 420∕4 = 105, M1 = 420∕3 = 140, M2 = 420∕5 = 84, M3 = 420∕7 = 60. We solve y0M0 ≡ 1 mod m1, d.h. -y1 ≡ 1 mod 4. The smallest non-negative solution is y0 = 1. We solve y1M1 ≡ 1 mod m1, d.h. -y1 ≡ 1 mod 3. The smallest absolute Solution is y1 = -1. We solve y2M2 ≡ 1 mod m2, d.h.-y2 ≡ 1 mod 5. The smallest absolute Solution y2 = -1. Finally we solve y3M3 ≡ 1 mod m3, d.h.-y3 ≡ 1 mod 7. The smallest non-negative solution is y3 = 2. Therefore, we obtain x ≡ 210 - 140 - 336 + 360 ≡ 94 mod 420. A solution of the simultaneous congruence is x = 94.

Note that in the described algorithm, the numbers yi and Mi does not depend on the numbers ai. If the values yi und Mi are calculated, then you can use the formula x = (∑ i=0naiyiMi) mod m , to solve the simultaneous congruence for every values of ai .
