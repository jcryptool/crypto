Signal-Protokoll

In diesem Plugin wird das Double-Ratchet des Signal-Protokolls visualisiert.
Insgesamt besteht das Signal-Protokoll aus mehreren Schritten die hier in der Onlinehilfe erklärt werden.

Das Plugin lässt sich über das Menü Visualisierungen oder über den Krypto-Explorer im Reiter Visualisierungen starten.

Die JCT-Onlinehilfe zum Signal-Protokoll ist in drei Teile aufgeteilt:

  • Benutzung des Plugins [diese Seite]
  • Algorithmus
  • Ablauf

Benutzung des Plugins

Das Plugin wird über das Hauptmenü "Visualisierung" und den Menüeintrag "Signal-Encryption" gestartet. Danach erscheint ein Reiter "Signal-Encryption, der 2 Unterreiter besitzt. Deren Namen sind:

  • Allgemein
  • Double Ratchet

Im Folgenden werden die Inhalte der 2 Unterreiter beschrieben.

 

1. Unterreiter "Allgemein"

Im ersten Unterreiter erfolgt eine einleitende Erklärung können Sie die benutzten Parameter sehen und ändern, und die Schlüssel generieren, die dann im 2. Unterreiter zum Signieren und Verifizieren benutzt werden.

Der folgende Screenshot zeigt die Standardeinstellungen, und Schlüssel, die durch Drücken des Buttons "Schlüssel generieren" zufällig erzeugt wurden. Sie können Schlüssel auch von Hand eingeben, um Beispiele aus der Literatur zu testen.

Gruppierung "Parameter"

Im ersten Unterreiter können Sie die ff. Verfahrensparameter verändern und einsehen. Die Parameter sind im Dezimalsystem (Basis 10) angegeben.
